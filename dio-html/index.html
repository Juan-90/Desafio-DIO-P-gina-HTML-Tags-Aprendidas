Surgimento da Internet

(Voltar)

Criada pelo britânico Tim Berners-Lee, o acrônimo HTML significa HyperText Markup Language, traduzido para o português como Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web. Sem ele, o navegador não conseguiria exibir textos como elementos ou carregar imagens e outros conteúdos. Os hipertextos são conjuntos de elementos conectados, que podem ser palavras, vídeos, documentos, etc. Quando conectados, formam uma rede de informações que permite a comunicação de dados, organizando conhecimentos e armazenando informações. Ao visitar uma página simples na web, é possível perceber diferentes distribuições e tamanhos para títulos, parágrafos, imagens, vídeos e outros elementos. Essa estrutura é estabelecida através do HTML. No início da web, era comum encontrar sites contendo apenas textos e imagens simples, com uma estrutura básica e sem estilizações. Porém, atualmente, é muito raro encontrar sites que possuam apenas elementos HTML. Por isso, podemos considerar o HTML como o "esqueleto" da página. Além do esqueleto, é necessário ter o corpo. Para isso, temos as linguagens CSS e JavaScript, que, em conjunto com o HTML, formam a base para todos os websites atuais. Veremos mais adiante o significado dessas linguagens.


Funcionamento do HTML

(Voltar)

Um documento HTML, identificado pela extensão .html ou .htm, é lido pelo navegador, que então renderiza seu conteúdo para que o usuário final possa visualizá-lo. Esses arquivos podem ser exibidos em qualquer navegador (Google Chrome, Safari, Mozilla Firefox, entre outros). Normalmente, um site é composto por várias páginas HTML. Por exemplo, um website com três páginas (homepage, página de contato e página de produtos) terá ao menos três documentos .html distintos, um para cada página. O código HTML pode ser escrito em qualquer editor de texto, como o Bloco de Notas. Cada página é formada por uma série de tags (ou elementos), que são os blocos de construção das páginas. Esses blocos são usados pelo HTML para marcar o conteúdo, criando a hierarquia e a estrutura, e dividindo-o em seções, parágrafos, cabeçalhos, entre outros.


O que são Tags

(Voltar)

Utilizando qualquer editor de texto, como Sublime Text, NotePad++, Visual Studio Code ou até mesmo o Bloco de Notas, é possível criar um documento com a extensão .html, que será renderizado nos navegadores. Como mencionado anteriormente, esse documento é composto por uma série de tags. As tags são códigos que definem toda a estrutura da página, como tamanho, fonte, cores, quebras de linha, entre outros. A maioria dos elementos em um documento HTML possui uma tag de abertura e uma de fechamento. Existem também tags de estrutura única, como a tag que realiza a quebra de linha. Por exemplo, para escrever um parágrafo, usamos a tag "p". Escrevemos o parágrafo e, em seguida, fechamos a tag com "/p": html

Meu primeiro parágrafo.

Assim, ao salvar o arquivo com a extensão .html e abri-lo em um navegador, o parágrafo será exibido na tela do navegador.
