HTML

html é o que define que é uma página html, é a marcação do texto

Head

head pé carrega antes do corpo da página

Title

a tag title serve para mostrar o titulo da página na aba do navegador

Body

body é o que vai aparecer para o usuaro quado ele navegar pela páagina

i

tag i significa que o texto vai ficar em itálico

strong

tag strong transforma o texto em negrito

é possivel combinar ambos os estilos (negrito e itálico)

input

algums tags como por exemplo o input, que serve para inserir um campo de texto, não tem um fechamento

img

a tag img que serve para inserir uma imagem, também não tem fechamento

atributos são valores que podem mudar o comportamento da tag

id

id é um identificador de elemento

h1 até h6

tags h1 até h6 são titulos e sub titulos de um texto, quanto maior o número, menor o tamanho da fonte do título

p

tag p é o parágrafo , o corpo do texto, ela fecha em /p

blockquote

tag blockquote serve para fazer uma citação de algum autor

u

tag u (underline) serve para sublinhar um texto ou uma palavra

mark

tag mark serve para marcar um texto, ela colore o fundo do texto que será marcado

sup

tag sup serve para criar uma marca superior elevada, ao estilo 'marca registrada'

ol

tag ol serve para criar uma lista ordenada, ela precisa das tag filhas chamadas tag li

li

tag li é a tag filha da tag ol, essa tag pode ser encadeada com varias tags ol e li

ul

tag ul serve para criar uma lista não ordenada, também pode ter suas filhas li

link

link significa ligação de arquivos web

a

tag a seguida de 'href' é uma tag ancora que faz a ligação entre os arquivos, ou seja, linka as páginas

tag a pode ter o atributo 'target=blank' que serve para abrir o link em outra aba

o atributo title serve para mostrar o toltip, um balão com o texto do significado do title

hr

tag que traça uma linha para divisória de conteúdo
